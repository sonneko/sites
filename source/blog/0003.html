cssの呼び出し方

cssには書く場所があります。それについて、紹介します。

cssの書く場所には 3通りあり、おすすめは最後に解説する物です。

タグの中に、属性として記述する。

htmlタグの中に、属性としてcssを記述することができます。

<p style="color: red;">赤色になる</p>

赤色になる

一番おすすめしたくない方法です。なぜかというと、タグそれぞれにcssを書くと同じものが重複し、見栄えが悪くなるからです。

pタグのstyle属性に"color: red;"という値を入れています。

headタグ内に記述する

headタグ内のstyleタグ中に、記述する方法です。

<head>
    <style>
        p{
            color: red;
        }
    </style>
</head>
<main>
    <p>赤色になる</p>
</main>

赤色になる

これもまた、推奨しません。注意したいのはheadタグの中にしか、書いてはいけないことです。

cssファイルの中に記述する方法

推奨するやり方です。まずはcss専用のファイルを用意します。

そして、適用したいhtmlファイルのheadタグ内に以下を記述します。

<link rel="stylesheet" href="https://******.com/css/******.css">

href属性の値はcssのファイルへのリンクを書いてください。

そうすると、このhtmlファイルに指定したリンク先のcssが適用されます。複数指定することも可能です。

セレクタの書き方

セレクターとは？

セレクタとは、cssの効果を適応させる範囲の記述のことです。

[セレクタ]{
    [設定対象]: [値];
}

基本の書き方を紹介します。

  • タグの名前をそのまま
  • クラス名
  • 「または」の選択方法
  • 「かつ」の選択方法

タグの名前をそのまま使う

例えば、全てのh1タグに対して、適用させたいなら「h1」がセレクタになります。全てのタグについて、これが使えます。（全選択はmainなど）

p{
    color: red;
}
a{
    color: yellow;
}

赤色になったpタグの中身

黄色になったaタグの中身

黄色の文字はとても読みにくいですね。薄い色は文字の色に使わないようにしましょう。

クラス名

divタグの属性であるclassを使って、指定できます。構文は「.[クラス名]」です。"."（ピリオド）を忘れないようにしましょう。

.list{
    color: red;
}
<div class="list">
    <p>ここは適用される</p>
</div>

<p>ここは適用されない</p>

ここは適用される

ここは適用されない

「または」の選択方法

これまでに紹介したセレクタを","（コンマ）で繋ぐことで、それぞれのセレクタの要素に対して適用されます。これで、重複記述を減らすことができます。

p,h5{
    color: red;
}
<p>ここは適用される</p>

<h5>ここも適用される</h5>

ここは適用される

ここも適用される

「かつ」の選択方法

.content p{
    color: red;
}
<div class="content">
    <p>ここは適用される</p>
    <h5>ここは適用されない</h5>
</div>
<p>ここは適用されない</p>

ここは適用される

ここは適用されない

ここは適用されない

先ほどと同じように、空白で区切ることで指定のタグに含まれるタグのみに適用させることができます。

もっと、複雑な「〜でない範囲」や、「奇数（偶数）番目にある〜」のようなセレクタもあります。セレクタは使いこなせるようになりましょう。

色の書き方

cssを書いていくのあたって、色を記述することがよくあります。

色の記述はこのような時に使います。

h1{
    background-color: [色の記述];
}

多くのcssプロパティの[値]には数字・色・指定された文字列などが入ります。

ここではその中の色の書き方について、紹介します。

色の書き方には主に4種類あり、使い分けないといけません。

  • 色の名前をそのまま書くパターン
  • カラーコードと呼ばれる16進法の数字を入れる方法
  • 色の三原色のそれぞれの値を代入する方式
  • 1つものの前に透明度を含めた方式

色の名前をそのまま書くパターン

p{
    background-color: blue;
}

青になる

赤色なら「red」、青色なら「blue」などと、色の名前をそのまま代入する方法です。一番直感的に分かりやすいと思います。

背景色が青のような濃い色だと、文字がほとんど読めませんね。前回話した通り、背景色に濃い色は使わないようにしましょう。

カラーコードと呼ばれる16進法の数字を入れる方法

p{
    background-color: #F1DE9E;
}

色の名前がわからない、、、1

「#」の後に、カラーコードを入れるとその色を反映させることができます。

この方法では、一つ目の方法と違って、名前もないような（あるんだろうけど）様々な色を記述することができます。

色の三原色のそれぞれの値を代入する方式

p{
    background-color: rgb(241,222,158);
}

色の名前がわからない、、、2

方式は先ほどと全く同じです。先ほどは16進法で表現しましたが、これは10進法の数字を用います。（1から255）

「rgb」は「red,green,blue」の頭文字です。それぞれの数字も同じ順序で記述します。

透明度を含めた方式

p{
    background-color: rgba(241,222,158,0.3);
}

色の名前がわからない、、、そして透明

先ほどのものに、透明度αを加えたものです。
これで、透ける要素を作ることができます。cssには半透明にするプロパティもあるのですが、それだと子要素まで透明になってしますので、重宝します。

「rgba」は「red,green,blue,α」の意味です。

α（透明度）の値は0以上1以下です。

配置の記述

cssにおいて、色と同じように重要なのが、要素の配置です。先ほど説明した色の書き方と比べかなり難しいです。

まず、htmlには要素があります。要素とはタグに囲まれた範囲のことです。実際には見えないですが、きちんとその範囲の境界線が内部で定義されています。

cssのborderタグを使うと、境界線を実際に表示できるので分かりやすくなるのではないかと思います。

一つの要素の中には、その要素を含む親要素と、その要素が含む子要素があります。

自分と親要素の間隔は「margin（外部余白）」とよばれ

自分と子要素の間隔は「padding（内部余白）」と呼ばれます。

また、自分は親要素のpaddingを除いた範囲のなかにいるわけですが、その中でどの方向（左・中央・右）に寄せるのかを表すのが、「text-align」です。textと書いていますが、文章以外の要素（画像）などにも使えます。

これを利用すると、上側のmargin（margin-top）を大きな値に設定することで、そこに余白を作ることができます。

cssで使う単位には「px」「%」があります。pxは絶対的な長さを、%は親要素の長さとの比を表します。
他にもたくさん単位はありますが、ここでは紹介しません。プロパティ（borderやmargin、colorなどの設定対象のこと）によったら、使えないケースもあるので注意してください。

よく使うプロパティ
  • background-image：背景画像の指定（URL）
  • background-color：背景色の指定（色）
  • width：横幅の指定（長さ）
  • height：高さの指定（長さ）
  • margin：外部余白の指定（後で解説）
  • padding：内部余白の指定（後で解説）
  • text-align：範囲の中でどちらに寄せるか（方向）
  • color：フォントの色の指定（色）
  • font-size：フォントの大きさの指定（長さ）
  • font-family：フォントの種類の指定（種類の名前）
  • border：境界線の指定（後で解説）

基本的にはプロパティにつく値は一つです。（）のなかに入っている値を入力してください。

しかし、中には複雑な値を入力しないといけないこともあります。
それについて解説します。

marginとpaddingの使い方

margin/paddingの使い方には 3種類あります。

  • 「margin: [値];」
  • 「margin: [高さの値] [横幅の値];」
  • 「margin: [margin-top] [margin-left] [margin-bottom] [margin-right];」

一つ目は全方向に対して、marginを指定します。

2つ目は横方向と縦方向に対して、marginを設定します。

3つ目は全方向にそれぞれmarginを設定します。

margin-[方向]でも設定できますが、これを用いるとより簡単に書くことができます。
このような短く書くことを可能にするプロパティをショートハンドプロパティと呼びます。

なお、paddingでも全く同じ構文があります。

borderの使い方

構文は以下の通りです。

border: [線の太さ（長さ）], [線の種類（後で解説）], [線の色（色）];

borderもショートハンドプロパティです。なので、border-[方向]-[指定対象]というプロパティで別々に指定することができます。

[方向]には「top」「left」「right」「bottom」が入ります。

[指定対象]には「color（色）」「style（種類）」「width（太さ）」

また、線の種類は10通りありますが、ここではよく使うであろう4種類を紹介します。

  • none：何もなし（初期）
  • solid：実線
  • dotted：点線
  • double：2本線

他には、凹んだように見える線などがあります。
