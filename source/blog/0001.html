日常でも見るWebページ。ほとんどの人はよく見ているのに、作り方を知っている人は少ないのではないでしょうか？

Webページ制作は意外と簡単にすることができます。ここではサイトの材料であるhtmlのデータを作る方法について、解説しています。

完全無料で、自分のサイトを作る方法はここで解説しています。ぜひ併せてご覧ください。

1、Webページの仕組み

Webページはサーバーから情報を送ってもらい（後述）、その情報を各デバイス（スマホやパソコンなど）で表示することで、私たちはWebページを閲覧できるようにになっています。

Webページを見るためには、見たいページを公開しているサーバーを自分で探さないといけません。そのための仕組みをプロトコル（通信規約）と言います。本筋とはズレるので、ここでは解説しません。

これからはWebページを見てもらうためには何のファイルが必要なのかを解説していきます。

2、Webページの構成

Webページに必要なファイルは3つあります。
ズバリ、「html」と「css」、「JavaScript」です！

htmlはメインのファイルです。具体的には、文章を書くところです。他にも「cssとJavaScriptはこれを使ってね」と指定したりすることができます。
html無しではWebページを作ることはできません！
拡張子は「.html」です。

cssは文章・画像の文字の色や配置を記述します。cssなしでも、Webページを作ることはできますが、白黒で味気ないページになってしまうでしょう。
拡張子は「.css」です。

JavaScriptを使うと、動きのあるコンテンツを作り出すことができます。例えば、ボタンを押したら、色が変わったり、アニメーションが表示されたりする仕組みを作れます。
拡張子は「.js」です。

綺麗で使いやすいWebサイトを作るには「html」「css」「JavaScript」を全て使わないといけません。

3、htmlについて

htmlは書き方がほとんど決まっています。

<[タグの名前] [属性]=[値]>[文章]</[タグの名前]>

構文は「<[タグの名前] [属性]=[値]>[文章]</[タグの名前]>」です。

[タグの名前]には「h1」「p」「a」などタグの名前を
[属性]には「href」「class」などタグによって、決まっている属性の種類を
[値]には属性によって、決まった値を入れます。
[文章]は基本的には、ページに表示する文章が入ります。
これを理解したら、あとはタグを覚えるだけです。
タフはたくさんあり、そのうちの一部はほとんど使わないので基本のもの以外は存在を知っているだけで、大丈夫です。

例えば、「Amorphous」という見出しを作るには「1番目の見出し」という意味のh1タグを
「google検索する」というリンクを貼るには、「リンク」という意味のaタグの中のhref属性にhttps://www.google.comを指定します。

<h1>Amorphous</h1>
<!--Amorphousという見出し-->

<a href="https://www.google.com">google検索する</a>
<!--「google検索する」と表示されるhttps://www.google.comへのリンク-->

基本のタグの使い方は次の記事で解説します。

6、cssの基本の書き方

cssも書き方がほとんど決まっています。

構文を下に示します。

[セレクタ]{
    [設定対象]: [値];
}

例えば、h1という文章を書く領域の文字の色を赤色にするには...

h1{
    color: red;
}

のように書きます。

7、JavaScriptの書き方

残念ながら、JavaScriptにはhtmlやcssなどのように基本の書き方というものは存在しません。なので、JavaScriptについてもう少し詳しく紹介します。

JavaScriptはクライアント側で動作する言語です。このサイトもJavaScriptを使っていますから、今もあなたが今使っているデバイスがJavaScriptを実行しているということです。

逆に、サーバー側で動作する言語にはPHPなどがあり、「特定の人にしかhtmlを渡さない」という処理も書くことができます。これを使ったら、サインイン/アウトをするシステムを作ることできます。

JavaScriptは命令の種類も多く創造性に富んでいるのは事実ですが、その分とても難しい言語です。
また、とても厳格（少し間違えただけで、エラーになってしまう）です。
これを書いている筆者もまだまだ勉強中です。

今回のまとめ

  • Webページは「サーバー→デバイス」に情報が送られことで、実現されている。
  • 送られる情報には「html」「css」「JavaScript」がある。
  • htmlは文章を指定するメインのファイル
  • cssはデザインを指定するサブのファイル
  • JavaScriptは動きをつけるサブのファイル
  • html・cssの書き方は、おおかた決まっている。
  • JavaScriptはクライアント側の言語である。
  • サーバーsideの言語にはPHPなどがある。
